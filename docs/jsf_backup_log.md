# Just Sean Flows (JSF) - 프로젝트 백업 및 현황 문서

> **목적**: 현재까지 진행된 디자인 개편, 오디오 세팅, 인터랙션 로직 등의 현황을 기록하여 NotebookLM 및 향후 작업의 기준점으로 삼기 위함입니다. 업데이트 사항은 기존 내용을 덮어쓰지 않고 이력으로 계속 누적됩니다.

---

## 🌟 현재 프로젝트 현황 (Current Overview)

### 🎨 UI/UX 및 테마
초기의 '화려한 스팀펑크 및 네온' 스타일에서 벗어나, **다크하고 고급스러운 시네마틱(Cinematic Editorial) & 멀티버스** 컨셉으로 완전히 전환되었습니다.

- **글래스모피즘(Glassmorphism) 기반 세련된 레이아웃**: 반투명한 `backdrop-blur`와 은은한 그림자(`mix-blend` 모드)를 사용하여 고급스러움을 극대화했습니다. 넓은 데스크탑 화면에서도 UI 요소들이 흩어지지 않고 세련된 HUD 상자(`max-w-[540px]`) 안에 담기도록 설계되었습니다.
- **국가별 포스터 카드 (멀티버스 UI)**: 각 국가별(한국, 일본, 아랍, 스페인 등 총 8개국) 고화질 사진 이미지가 배경으로 깔린 세로형 포스터 카드가 렌더링됩니다.
- **시네마틱 오프닝 (Cinematic Opening)**: 
  - `JUST.SEAN.FLOWS` 스택 타이포그래피 디자인으로 변경되어 럭셔리한 진입 감각을 선사합니다.
  - 백그라운드는 전체 화면을 시원하게 채우고, 텍스트와 진입 버튼은 고정된 HUD에 깔끔하게 배치됩니다.

### 🔊 오디오 세팅 (ElevenLabs 연동)
전체적인 사운드가 디지털/사이버펑크 전자음에서 **하이엔드 순수 어쿠스틱 및 클래식 기반**으로 100% 교체되었습니다.

- **시그니처 오프닝 사운드 (3D 공간음향)**: 진입 시 그랜드 피아노, 첼로, 플루트가 연주되는 입체 음향 재생.
- **점진적 홀드 사운드**: 드래그 앤 드롭 후 5초간 카드를 꾹 누르는 동안 신비로운 피아노 화음이 아르페지오로 이어집니다.
- **독점적 크로스페이드 BGM 룰**: 선택한 언어별 두 개의 전용 테마곡이 번갈아가며(Alternating) 자연스럽게 크로스페이드(Crossfade) 재생.
- **오디오 더킹(Ducking)**: AI 음성 안내 시 BGM 대역폭이 자동으로 낮아져 대사 전달력을 높입니다.

### ✨ 관리자 페르소나 "SEAN'S COMMENT"
플랫폼의 주인을 직접 투영하는 "SEAN"의 안내 시스템으로 전면 개편했습니다.

- **인터페이스 최상단 알림판 (MinaDirective)**: 
  - 세련된 스코어보드 디자인으로 화면 최상단에 렌더링됩니다. (PC에서 언어 그리드와 동일한 너비로 깔끔하게 정렬)
  - 확장/축소가 가능하며, 두 가지 페르소나(수석 지휘자 🎻, SEAN 🎙️) 모드가 상황에 맞게 전환됩니다.
- **글로벌 번역 및 다국어 지원**: 아랍어, 독일어, 일본어 등 8개 국어로 UI 환경 및 안내 코멘트가 100% 현지화되어 렌더링 됩니다.

## 📝 릴리즈 및 변경 이력 (Changelog)

### [2026-03-08] 오프닝 UI 리뉴얼 & 레이아웃 디버깅
- **시네마틱 오프닝 화면 전면 리뉴얼**:
  - 텍스트 길이가 과도하게 늘어지는 것을 방지하기 위해 `JUST.SEAN`과 `FLOWS`를 상하단 2줄로 배치하는 럭셔리 스택 타이포그래피 적용. 버튼을 누른 후 발동되는 섬광(Ignite) 연출에도 규격을 통일하여 텍스트 잘림 현상을 완벽 차단.
  - PC 모드에서 배경은 뷰포트 전체로 자유롭게 두되 텍스트와 UI 버튼 영역을 우아한 글래스모피즘(Glassmorphism) HUD 박스에 묶어 일관된 비율을 유지하도록 레이아웃 분리.
- **SEAN'S COMMENT 알림판 및 언어 그리드 규격 통합**:
  - `MinaDirective` 창 넓이와 `LanguageSelector` 메인 그리드 넓이를 동일하게 `max-w-[540px]`로 강제 매칭.
  - 크롬 100% 기본 폰트 사이즈 등에서 타이틀 글리치가 발생하여 두 줄로 쪼개지던 현상을 `whitespace-nowrap`을 씌워 완벽히 해결.
- **UI 번역 키 누락 버그 복구 및 렌더링 버그 제거**:
  - 5초 홀드가 완료된 후 다중우주(Awareness) 화면 진입 시 중앙 앵커 뒤에 보기 싫게 숨어 남아있던 'ALIGNED' 텍스트 패널(HUD)이 이벤트 전환 직후 완벽하게 사라지도록 로직 패치.

### [2026-03-08] Phase 1.5 ~ Phase 3 대규모 인터랙션 & UX 플로우 완성
- **경로 투표 (Phase 1.5)**: 
  - AWARENESS 닫기 액션 후, BGM 강제 압축(Ducking) 적용 및 화면 전체에 다크 글래스모피즘 오버레이 렌더링. 
  - TTS 엔진을 통한 "수석 지휘자"의 안내 음성 재생 (투표 vs 게임).
- **물리적 드래그 제약 & 스프링 효과 (Phase 2)**: 
  - 선택한 경로에 맞게 특정 빈 슬롯만 골드 글로우로 타겟팅(활성화)되며, 엉뚱한 곳에 놓으면 쫀득하게(Framer Spring) 원위치로 튕기는 Snap 효과 구현.
- **Glassmorphism 형태의 트렌디한 진행 바 (Phase 3)**:
  - 5초 홀드(HARMONIZING) 피드백 UI를 구식 사각형+소수점에서 최신 알약(Pill) 형태의 빛나는 프로그레스 바 + 퍼센테이지 단위(0~100%)로 완전 개편.
  - 전 세계 8개 국어에서 '조율 중/완료됨'을 자동으로 인식하여 렌더링하는 다국어 하드코딩 추가.
- **공사 중(WIP) 시스템 엔딩 (System Pause)**: 
  - 타겟에 안착하여 100% 도달 시 크래시를 방지하고, BGM을 다시 깎으며 "아직 해당 차원이 공사 중"이라는 위트 있는 TTS 개입 엔딩 연출 확립.

**⚠️ 향후 과제 및 주의사항**:

- **상호작용 체크**: 10초 대기 후 나타나는 Instagram 갤러리 모달 진입 시점과 모바일 화면 터치 혼선 주의 요망. 

### [2026-03-05] Harmonic Convergence 5초 홀드 인터랙션 패치
- **"Harmonic Convergence" 5초 홀드 UI**: 투박했던 기계식 게이지에서 하이엔드 깐느(Cannes) 시네마틱 스타일의 빛의 일렁임(Bloom)과 기하학적 다이아몬드 아티팩트가 회전하는 형태로 전면 개편했습니다. 
- **Pointer Capture 인터랙션 안정화**: 스마트폰 터치나 마우스 클릭 상태에서 미세한 떨림으로 인해 홀드가 취소되던 현상을 네이티브 `setPointerCapture` 이벤트를 통해 완벽히 해결했습니다.
- **무한 렌더링(Infinite Render Loop) 디버깅 완료**: 5초 홀드 성공 후 전환되는 `ComingSoonView`에서 숨겨진 칭호(Archetype)를 계산하며 발생하던 `Maximum update depth exceeded` 리액트 크래시를 상태 업데이트 최적화(`useCallback` 및 명시적 중복 방지 로직)로 영구적으로 수정했습니다.
- **UI 및 텍스트 디테일 마감**: SEAN'S COMMENT의 비활성화된 서브 페르소나 아이콘 밝기를 8% + 옅은 글로우로 세련되게 조정하였고, UI 알림 텍스트의 강제 줄바꿈(`\n`)을 제거하여 기기 너비에 맞게 유기적인 반응형 줄바꿈을 지원합니다. 또한 기록 보관소의 뱃지 타이틀이 각 언어별 딕셔너리에 맞게 정상 출력(Hydration)되도록 수정했습니다.

### [2026-03-08] AWARENESS 페이즈 시각적 집중 단속 및 UI 강제 정렬 패치
- **AWARENESS 초기 몰입감 극대화**: 진입 시 5, 9번 그리드 외의 비활성화된 그리드 투명도를 눈에 띄게 낮추고 추가적인 명암 처리를 통해 시선의 분산을 차단했습니다.
- **다국어 안내 문구 개선**: 진입 시 `SEAN'S COMMENT`의 영문 대기 메시지를 "Please read AWARENESS first to fully enjoy the experience." 로 구체화했습니다.
- **MinaDirective 상단 바 레이아웃 강제 정렬**: CSS Flex 속성을 무시하고 화면 폭에 상관없이 타이틀 텍스트와 아이콘이 무조건 왼쪽 가장자리에 밀착되도록 인라인 `absolute` 포지셔닝(`left: 24px`)을 하드코딩 방식으로 강제 적용해, 모바일/데스크탑 환경을 막론하고 타이틀의 좌측 정렬을 완벽하게 고정했습니다.
- **Harmonizing (운명 조율 중) 진행 상태바 중앙 정렬**: 언어 카드 하단부에 절대 좌표를 사용해 렌더링하던 프로그레스 바를 Flex 박스의 자동 마진(`mx-auto`) 기반으로 변경하여, 반응형 웹에서 부자연스럽게 우측으로 치우치던 현상을 잡고 투표 타겟 카드의 정중앙 하단에 아름답게 안착하도록 교정했습니다.
