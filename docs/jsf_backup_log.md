# Just Sean Flows (JSF) - 프로젝트 백업 및 현황 문서

> **작성일자**: 2026-03-04  
> **목적**: 현재까지 진행된 디자인 개편, 오디오 세팅, 인터랙션 로직 등의 현황을 기록하여 NotebookLM 및 향후 작업의 기준점으로 삼기 위함입니다. 사용자가 `jsf 대화 백업`을 요청할 때마다 갱신 또는 추가 기록됩니다.

---

## 1. 현재 UI/UX 및 테마 현황 🎨
초기의 '화려한 스팀펑크 및 네온' 스타일에서 벗어나, **다크하고 고급스러운 시네마틱(Cinematic Editorial) & 멀티버스** 컨셉으로 완전히 전환되었습니다.

- **글래스모피즘(Glassmorphism) 적용**: 강한 형광색 테두리를 없애고 반투명한 `backdrop-blur`와 은은한 그림자(`mix-blend` 모드)를 사용하여 고급스러움을 극대화했습니다.
- **국가별 포스터 카드 (멀티버스 UI)**: 기존의 단순한 그리드 버튼 형식에서 탈피하여, 각 국가별(한국, 일본, 아랍, 스페인 등 총 8개국) 고화질 사진 이미지가 배경으로 깔린 세로형 포스터 카드를 스크롤(Carousel)하는 형태로 변경되었습니다. 사진은 Unsplash를 통해 연동됩니다.
- **시네마틱 오프닝 (Cinematic Opening)**: 
  - 사이트 첫 진입 시 사용자가 올린 커스텀 콜라주 포스터(`intro_poster.png`) 배경이 중심이 됩니다. 
  - 배경 비율(contain)과 정중앙 정렬, 은은한 블러 처리를 기본으로 적용하고, 황금빛 애니메이션이 들어간 "CLICK TO ENTER" 게이트웨이가 고급스럽게 연출됩니다. 

## 2. 오디오 세팅 현황 🔊 (ElevenLabs 연동)
전체적인 사운드가 디지털/사이버펑크 전자음에서 **하이엔드 순수 어쿠스틱 및 클래식 기반**으로 100% 교체되었습니다.

- **시그니처 오프닝 사운드 (3D 공간음향)**: 
  - "CLICK TO ENTER" 버튼을 누를 때 넷플릭스 로고처럼 재생되는 5초 길이의 사운드입니다. 그랜드 피아노(중앙) + 첼로(좌측) + 플루트(우측)가 동시에 연주하며 아름다운 3D 공간음향(Binaural)을 냅니다.
- **지속적인 홀드 인터랙션 (Hold-to-Select)**:
  - 카드를 중앙 슬롯으로 드래그 앤 드롭한 뒤 "마우스를 꾹 누르고 있는 동안(Hold)" 진입 게이지가 차오르는 인터랙션입니다.
  - 홀드 중 **점진적인 신비로운 피아노 화음(낮은음 -> 중간음 -> 높은음 아르페지오)**이 추가되며 결제 완료 시 맑은 타격음 트랜지션이 터집니다.
- **독점적 크로스페이드 BGM 룰**:
  - 선택한 언어에 맞춰 2개의 전용 테마곡이 제공되며, 반복 선택 시 두 곡이 랜덤이 아니라 **번갈아가며(Alternating)** 재생되도록 고도화되었습니다.
- **오디오 더킹(Audio Ducking)**:
  - 안내 음성이 재생될 때 배경음악 볼륨이 서서히 줄어들고(Ducking) 멘트가 종료되면 복구됩니다. 

## 3. 글로벌 언어 현지화 및 "SEAN'S COMMENT" UI ✨
가상 비서 '미나(Mina)'를 삭제하고 플랫폼의 주인을 직접 투영하는 "SEAN"의 안내 시스템으로 전면 개편했습니다.

- **"SEAN'S COMMENT" UI 컴포넌트 안정화**:
  - 알림/경고/안내를 담당하는 `MinaDirective` 컴포넌트가 우측 상단이나 하단에서 시스템 메시지를 띄웁니다.
  - 텍스트 길이에 따라 좌우가 꿈틀거리는 현상을 막기 위해 디자인을 flex-1로 압착하여 상하로만 부드럽게 확장/축소(Fold)되도록 고정했습니다.
- **전역 UI 텍스트 완벽 번역**:
  - 아랍어(`تعليق شون`), 독일어(`SEANS KOMMENTAR`), 일본어(`ショーンのコメント`) 등 8개 국어로 "SEAN'S COMMENT" 타이틀과 "행동 개시" 본문이 100% 번역되어 렌더링 됩니다.
- **숨겨진 업적(Archetypes) 네이밍 변경**:
  - "미나의 모범 요원" -> "SEAN의 모범 요원" 등 모든 요소에서 시스템 세계관의 흔적을 션(Sean)에게 맞추었습니다.

## 4. 해결되지 않은 문제 및 향후 과제 (Homework) 📝

1. **'언어 선택' 이후의 진짜 컨텐츠 (Next Flow - 갤러리/업적 보고)**:
   - 언어 선택 이후 진입하게 될 갤러리(사진, 콘텐츠 등)의 전체적 오류 인프라 구축이 필요합니다. 사진/콘텐츠가 없을 때 알아서 채워 넣는 Fallback 로직 및 UI 배포 작업이 임박했습니다.
2. **모바일 크로스 브라우징 제스처**:
   - 모바일 환경에서의 조작, 폰트 스케일링, 홀드 터치 제스처 안정성 지속 모니터링이 필요합니다.
3. **오디오 애셋 성능 보완**:
   - TTS 프리로드(Preload) 및 모바일 메모리 한계 최적화가 중요 과제입니다.
