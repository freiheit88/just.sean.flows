# JSF 대화 백업 워크플로우

사용자가 `jsf 대화 백업` 이라고 입력하면, 이 워크플로우를 실행하여 현재 Just Sean Flows(JSF) 프로젝트의 현황(진행 상황, UI/UX 설정 상태, 해결되지 않은 문제 및 향후 과제)을 요약한 백업 문서를 생성합니다.

## 실행 절차

1. **상태 분석**: 현재 프로젝트의 주요 레이아웃, 오디오 설정(ElevenLabs 프롬프트 내용 등), 인터랙션 로직(Hold-to-Select 등), 테마(Multiverse, Cinematic 등)를 파악합니다.
2. **백업 파일 생성 및 업데이트**: 프로젝트 최상단 디렉토리의 `docs/jsf_backup_log.md` 파일에 현재 날짜와 시간을 기록하고, 분석한 상태와 향후 과제를 마크다운 형식으로 덧붙입니다. 파일이 없다면 새로 생성합니다.
3. **사용자 보고**: 요약된 내용을 사용자에게 `notify_user`로 전달하며, 백업이 완료되었음을 알립니다.
